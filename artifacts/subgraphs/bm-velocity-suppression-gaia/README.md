# BM Velocity Suppression Gaia Subgraph

This standalone Gaia package maps the selected LKM evidence root `gcn_7bca73ad98eb4ed4` into executable Gaia DSL. The subgraph covers the Bistritzer-MacDonald continuum-model suppression of the noninteracting Dirac-point velocity in twisted bilayer graphene near the largest magic angle.

Raw LKM evidence and discovery flags are copied under `artifacts/lkm-discovery/input/`; mapping, merge, and open-problem audit notes are in `artifacts/lkm-discovery/`.
